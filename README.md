# remote-ctrl

从任何设备控制电脑！

## 安装

1. 电脑端安装python3，下载并解压。
2. 电脑端运行安装依赖
   ```bash
   pip install -r requirements.txt
   ```
3. 电脑端运行程序
   ```bash
   python main.py
   ```

之后在其他设备(电脑、手机、平板等）打开浏览器，访问电脑端ip:23333即可控制！

## 支持手势

1. 左键单击，右键单击
2. 单指移动鼠标，单指按住后拖动鼠标
3. 双指滚动
